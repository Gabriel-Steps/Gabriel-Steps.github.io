Sobre Mim

Olá, meu nome é Gabriel Passos Francisco e sou desenvolvedor Full-Stack com habilidades sólidas em HTML, CSS e JavaScript, além de experiência em React para criar interfaces dinâmicas e responsivas. No back-end, tenho conhecimentos em Java, C#, Python e Node.js, que utilizo para desenvolver soluções robustas. Também possuo experiência no desenvolvimento de aplicativos móveis com React Native.


Atualmente, sou estudante do curso de Bacharelado em Ciências da Computação na Univali (Universidade do Vale do Itajaí). Tenho o hábito de desenvolver projetos práticos voltados para a programação web, aplicáveis no dia a dia, como, por exemplo, Landing Pages – páginas únicas projetadas para campanhas de marketing digital.


Já desenvolvi mais de 30 projetos de programação, todos disponíveis na minha plataforma do GitHub. Entre eles, destaque para aplicações web criadas com HTML5, CSS3, JavaScript e React, além de aplicações mobile para Android desenvolvidas com React Native. Também possuo experiência em back-end, utilizando linguagens como C# e Java.
